## Overview

A privacy-themed capture-the-flag game where players learn about privacy tools and cryptography by solving challenges. Educational gamification for privacy awareness.

## Features

- **CTF Challenges**: ZK proofs, encryption, steganography
- **Leaderboard**: Competitive privacy hacking
- **Learning Path**: Progressive difficulty
- **Rewards**: NFT badges for completion
- **Multiplayer**: Team-based challenges

## Challenges

1. **Steganography**: Hide messages in images
2. **ZK Puzzles**: Prove knowledge without revealing
3. **Crypto Cracking**: Break weak ciphers
4. **OSINT Privacy**: Find information leaks
5. **Smart Contract Auditing**: Find vulnerabilities

## Quick Start

```bash
npm install
cd challenges/level1 && npm start
```
